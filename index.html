HTML5 Semantic Elements

A Summary of What Semantic Elements Are

Semantic elements are HTML tags that are used to give meaning and structure to web page content beyond just its appearance.
These elements are important in web development because they provide context to the content and help search engines,
screen readers, and other assistive technologies to better understand the content of a web page.
Semantic elements include tags such as;

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <footer>
  • <figure>


These tags provide information about the type of content they wrap, making it easier for users to navigate a web page and understand its purpose.
For example, the <header> tag is used to define the top section of a web page, typically containing a logo, navigation menu, and other important information.
In short, semantic elements help to improve the accessibility, usability, and structure of web pages.

To learn more about semantic elements, what is acceptable and unacceptable, check out the Mozilla MDN homepage.
